# Sparse Addition

Adds two sparse_numeric vectors

## Usage

``` r
sparse_add(x, y, ...)

# S4 method for class 'sparse_numeric,sparse_numeric'
sparse_add(x, y)

# S4 method for class 'sparse_numeric,sparse_numeric'

```

## Arguments

- x:

  a sparse_numeric object

- y:

  a sparse_numeric object

- ...:

  additional arguments

- e1:

  a sparse numeric object

- e2:

  a sparse numeric object

## Value

A new sparse_numeric object that represents the sum of x and y
