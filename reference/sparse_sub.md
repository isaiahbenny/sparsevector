# Sparse Subtraction

Subtracts one sparse_numeric vector from another.

## Usage

``` r
sparse_sub(x, y, ...)

# S4 method for class 'sparse_numeric,sparse_numeric'
sparse_sub(x, y)

# S4 method for class 'sparse_numeric,sparse_numeric'
e1 - e2
```

## Arguments

- x:

  A sparse_numeric object.

- y:

  A sparse_numeric object.

- ...:

  Additional arguments.

- e1:

  a sparse numeric object

- e2:

  a sparse numeric object

## Value

A new sparse_numeric object representing the difference.

## Methods (by class)

- `sparse_sub(x = sparse_numeric, y = sparse_numeric)`: Subtract two
  sparse_numeric vectors
